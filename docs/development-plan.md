# 智答圈开发规划与团队分工

## 1. 项目概述

智答圈是一个面向高校学生和技术爱好者的智能问答与知识分享社区平台。系统以传统问答社区为基础，支持用户发布问题、回答问题、点赞收藏、标签分类、热榜排行和个性化推荐。

在扩展功能方面，项目计划接入 DeepSeek Flash 模型，实现基于工具检索的轻量级 Agentic RAG 智能问答能力。AI Agent 可以检索社区中的历史问题、回答和标签内容，并基于检索结果生成辅助回答或回答草稿。

## 2. 技术选型

| 层级 | 技术 |
|---|---|
| 前端 | Vue 3、Vite、Vue Router、Pinia、Axios |
| 后端 | FastAPI、SQLAlchemy、Pydantic、JWT |
| 数据库 | MySQL 或 PostgreSQL |
| AI 能力 | DeepSeek Flash API |
| 检索方式 | SQL 关键词检索、标签匹配、可选 BM25 |
| 接口文档 | FastAPI OpenAPI / Swagger |
| 协作工具 | Git、GitHub |

## 3. 功能模块划分

| 模块 | 功能说明 |
|---|---|
| 用户模块 | 注册、登录、JWT 鉴权、个人信息、积分等级 |
| 问题模块 | 发布问题、问题列表、问题详情、关键词搜索 |
| 回答模块 | 发布回答、查看回答、采纳最佳答案 |
| 标签模块 | 标签管理、问题标签绑定、按标签筛选 |
| 互动模块 | 点赞、点踩、收藏、浏览记录 |
| 排行模块 | 热榜、积分榜 |
| 推荐模块 | 基于标签、浏览历史和热度的简单推荐 |
| AI Agent 模块 | DeepSeek Flash 对话、社区内容检索、回答草稿生成 |

## 6. Git 协作规范

推荐使用以下分支：

```text
main                 稳定版本
dev                  日常集成分支
feature/auth         用户认证模块
feature/question     问题与回答模块
feature/frontend     前端页面模块
feature/ranking      标签、互动与排行模块
feature/ai-rag       推荐与 AI Agent 模块
```

开发流程：

1. 每个人从 `dev` 分支拉取自己的功能分支。
2. 功能完成后提交到个人功能分支。
3. 自测通过后合并到 `dev`。
4. 阶段性稳定后再由负责人合并到 `main`。

## 7. 提交规范

提交信息建议使用以下格式：

```text
feat: add user login api
feat: add question detail page
fix: correct ranking score calculation
docs: update api design
refactor: split ai service
```

常用类型：

| 类型 | 含义 |
|---|---|
| feat | 新功能 |
| fix | 修复问题 |
| docs | 文档修改 |
| refactor | 代码重构 |
| test | 测试相关 |
| chore | 工程配置或杂项 |

## 8. 验收标准

项目基础功能验收标准：

1. 用户可以完成注册、登录和退出。
2. 用户可以发布问题、查看问题列表和问题详情。
3. 用户可以回答问题，并由提问者采纳最佳答案。
4. 系统支持标签分类、搜索和筛选。
5. 系统支持点赞、收藏、积分和排行榜。
6. 系统可以根据热度、标签或浏览记录推荐问题。
7. AI Agent 可以基于社区内容检索结果生成辅助回答。
8. 前后端能够完整联调，核心流程可以稳定演示。
