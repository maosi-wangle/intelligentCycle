## 启动步骤

### 1. 启动后端服务
首先确保后端服务已启动：
```bash
cd d:\桌面\program\intelligentCycle\backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 启动前端开发服务器
```bash
cd d:\桌面\program\intelligentCycle\front
npm run dev
```

### 3. 访问应用
打开浏览器访问：`http://localhost:5173`
